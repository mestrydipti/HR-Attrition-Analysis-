# HR-Attrition-Analysis-
This repository contains an HR Attrition Analysis Dashboard created entirely in Microsoft Excel.  
The dashboard visualizes important HR metrics such as:

- Total Employees  
- Attrition Count & Attrition Rate  
- Gender Distribution  
- Age Group Insights  
- Department-wise Attrition  
- Job Role-based Attrition  
- Education-wise Attrition  
- Marital Status Breakdown  

This analysis helps identify trends, understand why employees leave, and support data-driven HR decisions.

 Tools & Features Used:

- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers  
- Conditional Formatting  
- Data Cleaning & Preparation  
- Data Visualization Techniques

 Purpose of This Project:

 The goal of this project is to help organizations:

- Understand the main factors behind employee attrition  
- Identify high-risk groups  
- Analyze demographic and departmental patterns  
- Support data-driven HR planning and retention strategies

Future Enhancements:

- Add monthly/yearly attrition trends  
- Create a Power BI version for deeper analysis
